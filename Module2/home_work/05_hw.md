## "Числа по порядку"

### Задание

Дано три целые числа. 

Упорядочите их в порядке возрастания.

![up](img/up.png)

### Формат входных данных

Даны три целые числа.

### Формат выходных данных

Вывести все числа от меньшего к большему.

#### Примечание

Предоставленную заготовку не менять. Ваша задача, написать алгоритм, который распределит значения трех переменных так, что в "a" будет наименьшее значение, а в "c" наибольшее.

### Решение задачи

```python
a = int(input("a: "))
b = int(input("b: "))
c = int(input("c: "))

if a > b and a > c:
    max_var = a
    if b > c:
        med_var = b
        min_var = c
    else:
        med_var = c
        min_var = b
elif b > a and b > c:
    max_var = b
    if a > c:
        med_var = a
        min_var = c
    else:
        med_var = c
        min_var = a
else: #c > b and c > b:
    max_var = c
    if a > b:
        med_var = a
        min_var = b
    else:
        med_var = b
        min_var = a

a = min_var
b = med_var
c = max_var

print(a, b, c)
```
